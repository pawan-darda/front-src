Fixtures can be labelled in the scenarios. You can filter the ones to display as follows

  • before GC will return all labels with before and GC in the name
  • An empty field will return all labels
  • -before GC will return all labels without before (note the minus sign) but with GC in the name
  • name:"start up diff before gc" (note the quotes) will return only the label start up diff before gc
